# mqtt2gridtracker

## Installation

### Install paho-mqtt

	pip install paho-mqtt


### Replace MYCALL, ip-address & port

    mycall = "MYCALL"
    myloc = "XXYY"
    host = "192.168.1.300"
    port = 2237


### Control 

Show received by other calls 

     showrec = True

Show received by you call

     showmyrec = True


pip install paho-mqtt
